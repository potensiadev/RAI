## Enterprise Plan Overview

Our Enterprise plan is designed for large organizations with custom requirements. Pricing is determined through individual consultation based on the following factors:

### Pricing Factors

| Factor | Description |
|--------|-------------|
| **Team Size** | Number of users requiring access |
| **Monthly Volume** | Expected resume analysis volume |
| **Integration Requirements** | API access, SSO, custom integrations |
| **Support Level** | Dedicated account manager, SLA requirements |
| **Contract Term** | Annual vs. multi-year agreements |

### Typical Enterprise Engagement

- **Minimum Team Size**: 10+ users
- **Minimum Contract**: 12 months
- **Custom Features**: Dedicated support, custom branding, API access, advanced analytics
