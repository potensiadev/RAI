## 2. 구독 취소 및 환불

### 2.1 14일 전액 환불 정책 (Paddle 호환)

> **중요**: 본 정책은 Paddle 결제 플랫폼 정책에 부합하도록 설계되었습니다.

| 기간 | 환불 정책 |
|------|----------|
| **결제 후 14일 이내** | **100% 전액 환불** (서비스 이용 여부 무관) |
| **결제 후 14일 경과** | 환불 불가, 현재 주기 종료까지 서비스 이용 가능 |

#### 환불 절차

1. 설정 → 구독 관리 → [구독 취소] 클릭
2. 14일 이내인 경우 자동으로 전액 환불 처리
3. 환불 완료까지 5~7영업일 소요 (원결제 수단)

#### 취소 후 서비스

- 잔여 크레딧: 결제 주기 종료일까지 사용 가능
- 데이터: 취소 후 30일 보관, 이후 삭제

## 3. 크레딧 환불 정책

### 3.1 시스템 오류로 인한 크레딧 환불

| 상황 | 환불 | 처리 |
|------|------|------|
| 업로드 실패 (서버 오류) | 자동 | 즉시 |
| 분석 실패 (AI 오류) | 자동 | 즉시 |
| 중복 차감 | 자동 | 즉시 |
| 분석 품질 불량 | 자동 | 즉시 |

#### 분석 품질 불량 자동 환불 기준

**자동 환불 조건 (모두 충족 시):**

| 조건 | 기준 |
|------|------|
| 신뢰도 점수 | 0.3 미만 |
| 핵심 필드 누락 | 이름, 연락처, 경력 중 2개 이상 추출 실패 |

**처리 방식:**

```
분석 완료 시 시스템이 자동 판단:
1. confidence_score < 0.3 확인
2. 핵심 필드(이름, 연락처, 경력) 중 2개 이상 누락 확인
3. 조건 충족 시:
   - 크레딧 자동 환불
   - 사용자에게 알림: "분석 품질이 기준에 미달하여 크레딧이 자동 환불되었습니다."
   - 분석 결과는 참고용으로 제공 (일부 추출된 정보 확인 가능)
```

**자동 환불 제외:**

- 원본 이력서 자체에 해당 정보가 없는 경우
- 비표준 형식 이력서 (포트폴리오, 자기소개서 등)
- 외국어 이력서 (현재 한국어만 지원)

> 자동 환불 외 품질 관련 문의: support@rai.com

| 상황 | 처리 방식 |
|------|----------|
| 지원되지 않는 파일 형식 | **업로드 차단** (크레딧 미차감) |

## 6. 환불 절차

### 6.1 환불 신청

```
설정 → 구독 관리 → [환불 신청] 버튼
```

### 6.2 처리 기간

| 유형 | 처리 |
|------|------|
| 7일 이내 전액 환불 | **즉시 승인** → 3~5영업일 입금 |
| 부분 환불 | **즉시 계산** → 5~7영업일 입금 |
| 크레딧 환불 | **즉시 적용** |

### 6.3 환불 수단

| 원결제 수단 | 환불 방식 |
|------------|----------|
| 신용카드 | 카드사 부분 취소 |
| Paddle (해외결제) | Paddle Refund API |

#### Paddle 부분 환불

```
POST https://api.paddle.com/transactions/{transaction_id}/refund
{
  "amount": "35300",
  "reason": "partial_refund_prorata"
}
```

**처리 흐름:** 환불 신청 → 금액 계산 → Paddle API 호출 → 원결제 수단 환불 → 이메일 통보

**제한:** 원거래 금액 초과 불가, 거래 후 180일 이내
